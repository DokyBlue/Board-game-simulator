# 德州扑克规则说明（项目实现基线）

> 说明：由于当前仓库未包含“规则附件”原文，本实现采用标准 No-Limit Texas Hold'em 的通用规则作为第一版基线，可按你的附件进一步微调。

## 1. 基本流程

1. 每位玩家获得 2 张底牌（仅自己可见）。
2. 公共牌分三轮发放：
   - Flop：3 张
   - Turn：1 张
   - River：1 张
3. 玩家用“2 张底牌 + 5 张公共牌”中任意 5 张组成最大牌型。
4. 比较牌型，确定赢家并分配底池。

## 2. 盲注

- 小盲注（SB）和大盲注（BB）由指定位置玩家强制投入。
- 当前实现默认：
  - `smallBlind = 10`
  - `bigBlind = 20`

## 3. 牌型大小（高到低）

1. 皇家同花顺：A-K-Q-J-10 且同花色
2. 同花顺：任意连续 5 张且同花色
3. 四条：4 张同点数
4. 葫芦：3 张同点数 + 2 张同点数
5. 同花：5 张同花色
6. 顺子：5 张连续点数
7. 三条：3 张同点数
8. 两对：2 + 2 + 单张
9. 一对：2 张同点数
10. 高牌：以上都不满足

## 4. 同牌型比较

- 先比较牌型等级。
- 同等级时比较关键点数（例如三条点数、最大对子点数等）。
- 仍相同时比较踢脚牌（Kicker）。

## 5. 项目当前实现差异

当前为快速可运行原型，做了以下简化：

- 下注流程简化为“自动盲注 + 直接摊牌比较”，暂未开放逐轮玩家操作。
- 暂未处理平分池与复杂边池分配。
- Bot 为静态玩家，不含策略行为。

后续可在此基线补齐完整对局状态机与联机逻辑。
