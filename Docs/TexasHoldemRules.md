# 德州扑克规则与当前实现说明

## 1. 对局人数

- 单人模式：1 人类玩家 + 5 Bot。

## 2. 每局流程

1. 发放每人 2 张底牌。
2. 下盲注（SB/BB）。
3. 进入 Preflop 下注轮。
4. 发 Flop（3 张公共牌）并下注。
5. 发 Turn（1 张公共牌）并下注。
6. 发 River（1 张公共牌）并下注。
7. 摊牌比较，赢家获得底池。

## 3. 玩家动作

- **Call**：补齐到当前轮次最大下注。
- **Raise**：在 Call 后继续加注（当前为固定步长）。
- **Fold**：弃牌，不再参与本局。

## 4. Bot 策略（本项目要求）

- 所有 Bot 统一执行自动跟注（Call），不主动加注，不弃牌。
- 最后进入摊牌比较。

## 5. 牌型比较

项目实现支持完整牌型比较：

1. 皇家同花顺
2. 同花顺
3. 四条
4. 葫芦
5. 同花
6. 顺子
7. 三条
8. 两对
9. 一对
10. 高牌

同牌型按照踢脚牌（Kicker）比较。
